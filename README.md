## Repository Structure

- `aime_2024/` -- Complete AIME 2024 problem set (30 problems) with solutions
- `irrationality_of_sqrt2/` -- Lean 4 verified proof of sqrt(2) irrationality
- `chain_circles_inradius.md` -- Tangent circles chain problem
- `sum_of_cubes/` -- Sum of first n cubes equals square of sum (Lean 4)
- `n_cube_sub_n_divisible_by_six/` -- n³ - n is divisible by 6 (Lean 4)
- `sum_of_odds/` -- Sum of first n odd numbers equals n² (Lean 4)
- `squares_mod_three/` -- Squares modulo 3 are 0 or 1 (Lean 4)
- `sq_sum_ge_two_mul/` -- a² + b² ≥ 2ab for natural numbers (Lean 4)
- `octagon_probability.lean` -- Octagon probability problem
